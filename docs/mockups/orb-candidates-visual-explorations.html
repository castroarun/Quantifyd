D · Conviction heat-row (recommended)

Classical table kept — but each row's background tint reflects conviction: broken-out rows get a strong directional gradient (green for long, red for short), watching rows get amber if close to trigger, blocked rows fade. A "conviction chip" on the right distills all filters into a single signal. Keeps density, adds a visual hierarchy that says "look here first."
