Which one?

My picks in order:

  1. A (Heat strip) — most information-dense, closest to what's live, minimal API changes.
  2. E (Sparklines) — the most "interesting" visually and adds real trading context (shape of the move), but needs a backend addition (52w daily closes per holding).
  3. C (Buckets + chips) — the most dashboard-feeling; great when you have 20+ holdings to sort through.
  4. B (gauges) and D (scatter) are nice but niche — B takes space, D is advanced.
