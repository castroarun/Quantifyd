Pine Script — TradingView Setup

How to Use

  1. Open TradingView → Pine Editor → New Indicator
  2. Paste the code from pinescripts/nondirectional_range_detector.pine
  3. Apply to BANKNIFTY daily chart
  4. When you see the green diamond signal:
    • Check the info table (top right) for exact CE and PE strikes
    • The orange lines are your short strikes (SELL)
    • The purple dotted lines are protection wings (BUY for iron condor)
    • The green shaded zone is the expected range
  5. Set up alerts: right-click indicator → "Add Alert" → select "Range Signal"

Strategy Execution Playbook

Entry Rules

  • Apply indicator on BANKNIFTY daily chart
  • Wait for green diamond signal (BB Squeeze active)
  • Enter bi-weekly short strangle: Sell CE + PE at strikes shown in table
  • Entry only on signal day — no chasing next day
  • Minimum 10 bars gap between trades (no overlapping positions)

Position Sizing

  • 5 lots per ₹10L capital (best risk-adjusted: 22.2% CAGR, 1.9% DD)
  • 3 lots for conservative sizing (14.0% CAGR, 2.2% DD)
  • BankNifty lot = 15 units (5 lots = 75 units)
  • Monthly expiry options (30 DTE) — BankNifty weeklies no longer available

Exit Rules

  • Primary: Hold to expiry (let theta do the work)
  • Stop loss: Exit if either leg goes 2x premium collected
  • Early exit: Close at 80% of max profit (premium captured)
  • Adjustment: If one strike is breached, consider rolling the untested side closer

Risk Management

  • Never sell strangles on event days (RBI policy, Budget, Elections, earnings)
  • Avoid first 2 days and last day of weekly expiry
  • If VIX > 20, skip the trade or go with iron condor instead
  • Maximum 1 open position at a time
